# AI-Based SDG Mapping for ESG Disclosures
This project analyzes how corporate sustainability disclosures align with the United Nations Sustainable Development Goals (SDGs).
The analysis uses Natural Language Processing techniques to map sustainability report text against SDG targets.

## Methodology
- Python-based NLP analysis
- Sentence Transformers for semantic similarity
- TF-IDF similarity scoring
- ESG disclosure dataset

## Objective
To demonstrate how AI-driven text analysis can improve transparency and analytical rigor in ESG reporting.

## Tools Used
Python  
NLP  
Sentence Transformers  
TF-IDF Similarity
